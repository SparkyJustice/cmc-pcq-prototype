Upload your witness statements

All witness statements must:

  • start with the name of the case and the claim number - 'Clark v Richards. Claim number: 352MC733'
  • include the witness's full name and address
  • use numbered paragraphs and numbered pages
  • end with this paragraph: 'I believe that the facts stated in this witness statement are true.'
  • be signed and dated by the witness

You can download and complete a witness statement template (DOCX, 123KB). When you've completed it, upload the file on this page.

You can upload as many files as you need.
