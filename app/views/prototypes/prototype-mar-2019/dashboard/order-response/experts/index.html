Propose 3 experts

You must propose 3 independent experts - the {% if data['party'] == 'defendant' %}claimant{% else %}defendant{% endif %} will then pick one.

You must not discuss the case with the experts before they're appointed.

How to choose your experts

Make sure the expert you choose meets these conditions:

  • they know what an expert in a money claim does - ideally they've done it before
  • they can write a report about technical details of the case which answers questions from both parties
  • they're not a friend or a relative
  • the cost isn't out of proportion compared to the total value of the claim

How to use the expert

Both you and the {% if data['party'] == 'defendant' %}claimant{% else %}defendant{% endif %} can ask questions. The expert will answer these in a report. The judge can use the report to make a ruling.

Who pays for the expert

You and the {% if data['party'] == 'defendant' %}claimant{% else %}defendant{% endif %} share the cost of hiring the expert.

If you win the case you may be able to recover up to Â£750 of this cost. A judge will rule on this at the end of the hearing.
